# React Native

Reactの文法でネイティブアプリを作ることができるFacebook製のOSS。  

# 使い方
```console:
react-native init todolist # アプリの雛形を作成
react-native run-ios
```
# デバッグ
* Ctrl+DでDevelopmentメニューを起動→ ブラウザからRemode Debugが可能。
