# docker

コンテナAPI。  

* なぜDockerか?  
同じアプリケーションの複数バージョン, 複数インスタンスを同時に走らせられるため。  

# 使い方
$ sudo apt-get install docker.io # インストール
$ sudo docker images # イメージの確認
